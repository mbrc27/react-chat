react-chat
===
React, redux, redux-saga chat app


1. Setup
- Install using yarn or npm
```
yarn install
```
- Add firebase setup to src/db/config.json
```json
{
    "apiKey": "",
    "authDomain": "",
    "databaseURL": "",
    "projectId": "",
    "storageBucket": "",
    "messagingSenderId": ""
}
```
- Run using create-react-app scripts
```
yarn start
```
- Tests
```
yarn test
```
